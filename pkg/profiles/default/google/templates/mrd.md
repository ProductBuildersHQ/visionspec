## OKR Scoring Guide

### Key Result Types

| Type | Description | Example |
|------|-------------|---------|
| **Metric** | Quantitative measurement | "Reduce p99 latency from 500ms to 200ms" |
| **Milestone** | Binary achievement | "Launch feature X to 100% of users" |
| **Health** | Maintain while changing | "Maintain 99.9% availability during migration" |

### Scoring

| Score | Meaning |
|-------|---------|
| 0.0 - 0.3 | Failed to make progress |
| 0.4 - 0.6 | Made progress but fell short |
| 0.7 - 0.9 | Delivered (this is success!) |
| 1.0 | Fully achieved (maybe sandbagged?) |
